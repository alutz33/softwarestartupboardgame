# Persona Card Portrait Prompts

Style: Modern pixel art caricature portraits. Detailed pixel art with rich color gradients and smooth shading, character select screen aesthetic, head and shoulders only, vibrant colorful palette on dark cosmic/themed background, clean pixel edges, no text. Vertical 3:4 aspect ratio (768x1024).

Save each image as a PNG using the filename listed. Place them in this directory (`public/personas/`).
